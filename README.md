# StudyMate - CMPE382 Operating Systems

TED Universitesi CMPE382 Operating Systems dersi icin interaktif calisma uygulamasi.

## Kurulum

```bash
npm install
npm run dev
```

Tarayicida `http://localhost:5173` adresini ac.

## Calisma Modlari

- **Konu Ozeti** - Turkce anlatim, Ingilizce terimler, analogiler, diyagramlar
- **Flashcard** - 3D kart cevirme, zorluk derecelendirme
- **Quiz** - Coktan secmeli, aciklamali geri bildirim
- **Bosluk Doldurma** - Ipuclu, coklu cevap kabul
- **Sinav Pratigi** - Hesaplama, algoritma tamamlama, dogru/yanlis (gercek sinav formatinda)

## Ozellikler

- XP / Level / Streak sistemi
- Pomodoro zamanlayici (25dk calisma / 5dk mola)
- Klavye kisayollari (Space, 1-4, Enter)
- Dark tema
- Ilerleme LocalStorage'da kaydedilir

## Teknoloji

React + Vite + Tailwind CSS
